# PGN Steganography & Manipulation Tools

Цей проєкт призначений для аналізу, модифікації та генерації стелс-модифікацій PGN (Portable Game Notation) файлів для шахів із точним збереженням вихідного розміру файлу в байтах.

---

## 📂 Структура файлів проєкту (File Roadmap)

### 1. [`fix_notation.py`](file:///c:/Users/dmytr/PycharmProjects/PythonProject10/fix_notation.py)
* **Призначення:** Генерує 30 варіантів файлу `game.pgn`, замінюючи один конкретний символ на візуально схожий (омогліф) або вносячи мікро-зміну.
* **Як працює:**
  * Використовує кодування `cp1251` (де 1 символ завжди дорівнює 1 байту).
  * Робить точкову заміну 1-в-1 (наприклад, англійську літеру `a` на кириличну `а`, пробіл на нерозривний пробіл, або `1-0` на `0-1`).
  * Перевіряє розмір кожного файлу, гарантуючи, що він з точністю до байта відповідає оригіналу.
* **Результат:** Зберігає файли у папку `fixed_pgns/` з назвами від `fixed_v1.pgn` до `fixed_v30.pgn`.

### 2. [`truncate_pgn.py`](file:///c:/Users/dmytr/PycharmProjects/PythonProject10/truncate_pgn.py)
* **Призначення:** Обрізає шахову партію на певній кількості ходів (від 40% до 97%), зберігаючи оригінальний байтовий розмір файлу.
* **Як працює:**
  * Парсить теги (headers) та ходи.
  * Видаляє частину ходів в кінці партії.
  * Додає необхідну кількість пробілів у тег `[Event "..."]` як заповнювач (padding), щоб компенсувати видалені ходи.
* **Результат:** Файли з обрізаними ходами мають абсолютно ідентичний розмір файлу з оригіналом.

### 3. [`truncate_stealth.py`](file:///c:/Users/dmytr/PycharmProjects/PythonProject10/truncate_stealth.py)
* **Призначення:** Альтернативний скрипт для обрізання PGN з використанням прихованого доповнення довжини.

### 4. [`write_test_pgn.py`](file:///c:/Users/dmytr/PycharmProjects/PythonProject10/write_test_pgn.py)
* **Призначення:** Допоміжний утилітний скрипт для швидкого створення тестового PGN файлу.

---

## 🛠️ Як запускати

1. Переконайтеся, що в корні проєкту лежить файл `game.pgn`.
2. Запустіть потрібний скрипт через термінал:
   ```bash
   python fix_notation.py
   # або
   python truncate_pgn.py
   ```
3. Усі згенеровані файли з'являться в папці `fixed_pgns/`.
